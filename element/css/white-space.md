# CSS white-space 属性

### 实例
规定段落中的文本不进行换行：
```css
p{
    white-space:nowrap;
}
```

### 定义和用法
white-space属性设置如何处理元素内的空白。
这个属性声明建立布局过程中如何处理元素中的空白符。值pre-wrap和pre-line是CSS 2.1中新增的。
|默认值：|normal|
|---|---|

|继承值：|yes|
|---|---|

|版本：|CSS1|
|---|---|

|JavaScript：|object.style.whiteSpace="pre"|
|---|---|
