# arguments.callee

argument.callee属性包含当前正在执行的函数。

## 描述
`callee`是`arguments`对象的一个属性。它可以用于引用该函数的函数体内正在执行的函数。这个函数的名称是未知时很有用，例如在没有名称的函数表达式(也称为"匿名函数")内。
